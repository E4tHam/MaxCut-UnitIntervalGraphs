
Help

    help                                Gives this page


Graph Source
    
    -r, --random                        Randomize the Unit Interval Graph

    -fi, --file-in <Input file path>    Graph is loaded from file


File Cut

    -fc, --file-cut                     Cut according to file input


Graph Structure

    -b, --basic                         No restrictions on structure

    -p, --path                          Path structure

    -s, --staircase                     Staircase structure


Max-Cut Algorithm

    -bf, --brute-force                  Run brute-force algorithm

    -eo, --every-other                  Run every-other approx algorithm

    -si, --sifferman                    Run Sifferman alg (only on path)


File Out Path

    -fo, --file-out <Output file path>  Change the path of the output file
